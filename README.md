# 🛰️ AGRImotus — Space-Water Nexus
**CASSINI Hackathon 2026 - Challenge 1: Equitable Water Access**

## 🚀 Overview
AGRImotus is a satellite-powered platform designed to optimize water distribution in agricultural zones. 
By merging **Copernicus Sentinel-3** thermal data with proprietary **Edge AI** models, we provide real-time insights into soil water stress.
